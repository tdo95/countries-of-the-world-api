Documentation

Countries of the world API is a database that stores social and geogrpahical information about all the soverign nations in the world and some other states. Territores like Greenland and Puerto Rico are not yet included this collection. All information within this API is scraped from Wikipedia. If you would like to contribute please visit here.

ENDPOINTS

GET https://countries-of-the-world-api-production.up.railway.app/api/:name

Retrieves an array list of country objects whose name match the query parameter string

GET https://countries-of-the-world-api-production.up.railway.app/api

Retrieves entire API resource, with each country object listed within an array.

PROPERTIES

PROPERTY DESCRIPTION
countryName string Name of country, standard casing
wikiLink string Link to country information on wikipedia
capital string Capital city of the country, if none string will be empty
officialLanguages array List of all official languages in country, if none array will be empty
population string Estimate of population as of 2022. Numbers in string are comma seperated
gdp object Gross Domestic Production of the country. Object contains pppTotal property which reflects the country's total GDP standardized for inflation and to be used for comparisons of living standards between different countries
gini string Gini coefficient, measure of income distribution, and is a representation of income inequality within a nation. The higher the value, the greater the divide
currency string Name of currency used within the nation
drivingSide string Side of the road driven on within the country. Either left or right
flag string Link to flag image. 125px by 75px

*Note: Some countires may not have all property data. Unavailable data properties are represented by an empty value. e.g. empty string values are '', empty array values are [].
